Model <---- Controller ----> View

View - Whatever is visible to the user

Controller - The Coding which would act between model and controller

Model - How the Application behave between model and the controller

controller receive the request from the view(user) and pass it to the model after doing somework with model return it to controller.Then controller may or may not modify the data then give it to the view
